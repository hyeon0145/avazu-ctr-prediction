# Avazu CTR Prediction

This project uses the following methods:
- Feature Hashing
- Logistic Regression
- Stochastic Gradient Descent

For detailed information: https://www.kaggle.com/c/avazu-ctr-prediction

## Environment
- Java 7
- Hadoop 1.2.1

## Usage
1. Edit `build.xml`
2. Build the project
3. Edit `run.sh`
4. Execute `run.sh`
